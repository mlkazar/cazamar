Search Examples
college rock
Search for stations with both 'college' and 'rock' as keywords

electronic c:fr
Search for electronic music stations in France. Can use any standard 2 character country code.

g:jazz new orleans
Search for Jazz stations with New Orleans as keywords

Note that if searching for a specific station name, rotate the picker at the top to "By name" instead of its default "By keyword." Also note that you can use "t:" for city specific searches, as in "t:chicago" to search for stations in Chicago.
