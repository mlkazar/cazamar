Channel Sources

This screen shows all of the sources for your audio channel. Sources can be removed by swiping left on the source and selecting the exposed "Delete" button. The "Clear All" button removes all sources.

Pressing the "+UPNP" button lets you configure your UPNP / DLNA servers. If you intend to play or download music from your UPNP / DLNA servers, you must configure the servers first, which loads the music names and locations into the AudioGalaxy library. Afterwards, you can simply add music from UPNP sources from the Source menus.

At the bottom are the following control buttons:

  • +Sources
    • Add or remove media groups, such as playlists, albums, or radio stations, to the channel. Switches to Add Source screen, described below.
  • Config UPNP
    • Switches to UPNP configuration screen, where you can discover UPNP servers and load the names of the music found there into the library.
  • Clear All
    • Removes all sources from this channel.
  • Done
    • Finishes adding sources, and returns you to the Song List screen, where you can reorder songs or remove individual songs from the channel.
  • Help
    • Display this screen.

Adding Sources to Channels

When you press +Sources, the Add Source screen appears, allowing you to select media sources to include in your channel. Note that the iTunes library and UPNP libraries are separate, and have separate Add buttons. Media you can select include:

  • +Playlists / iTunes
    • All songs in an iTunes playlist.
  • +Playlists / UPNP
    • All songs on any UPNP server.
  • +Artists / iTunes
    • All songs in the iTunes library by a specific artist.
  • +Artists / UPNP
    • All songs on a UPNP server by a specific artist.
  • +Albums / iTunes
    • All songs in a specific album in the iTunes library.
  • +Albums / UPNP
    • All songs in a specific album on a UPNP server.
  • +Songs / iTunes
    • An individual song from the iTunes library.
  • +Songs / iTunes
    • An individual song from a UPNP server.
  • +Genre
    • All songs in the iTunes library with a specific genre.
  • +Genre
    • All songs on a UPNP server with a specific genre.
  • +Podcasts
    • All shows in a podcast's RSS feed. You must use the search box to locate podcasts. Also note that individual podcasts must be downloaded before they can be played (swipe left in main editing screen).
  • +Radio
    • An Internet radio station.

When adding a music item, you can select a minimum star rating filter. You can also press the "Cloud" button to include media items from your iTunes library cloud, but not present on your iPhone.

If you press the information button "Circle-i", instead of selecting that playlist, artist or album, AudioGalaxy will zoom into a new selection screen showing only the songs in that group. You can then select specific items from that group.

You can remove a media entry by swiping the entry left, and pressing the exposed DELETE button.

Podcasts / RSS Feeds

After adding Podcasts media groups to a channel, you must select specific podcasts to download in the Song List screen. Red-tinted podcasts haven't been loaded, while green tinted ones have. Swipe left on a podcast to load or unload a podcast, or to see the detailed description of the podcast. When playing podcasts, the player will move from one downloaded podcast to the next, ignoring unloaded podcasts.
