Searching for Stations

You enter the search system by selecting 'Add Station" from the main menu button on the main screen. You can do two types of searches. "Name" searches treat words you're searching for as words in the station name. "Genre" searches treat the words you provide as keywords in the station database records.

The search system examines several online databases to resolve a station name. These include radio-browser.info, radiotime.com, shoutcast.com, streamtheworld.com and dar.fm.

In addition to the basic treatment of words in a search query, you can precede a word with a tag which will force the search system to treat the word specially in those databases that provide more powerful searches. Specifically, prefixing a word with "t:" will interpret the rest of the word as the name of a city. Prefixing a word with "c:" means the rest of the word is the two character country code for the station. Prefixing the word with "g:" indicates that the rest of the word should be interpreted as a musical genre. So, for example, doing a name search with the following string

jazz c:fr

will return all Frencch stations with "jazz" in their name. If you select "By Keyword" instead of "By Name" in the drop down menu at the top of the search page, it will return all French stations with "jazz" as a keyword in any station database.

Once the stations have been located, RadioStar verifies that they're working at the moment, and adds the stations with at least one working stream to the table it displays. Next, the user should examine the matched stations, and tap some of their names to select which stations should be added to RadioStar's set of stations. Finally, click the "Done" button at the bottom of the screen, which will return to the updated station list page.
